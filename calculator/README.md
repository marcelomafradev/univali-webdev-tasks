# Calculator

Calculadora desenvolvida com HTML, CSS e JavaScript.

## Funcionalidades

- Operações: adição, subtração, multiplicação, divisão e porcentagem
- Preview do resultado enquanto digita
- Suporte a teclado (números, operadores, Enter, Backspace, Escape)
- Tratamento de erros (divisão por zero, expressão inválida)

## Arquivos

- `index.html` → estrutura da página
- `style.css` → estilos
- `script.js` → lógica da calculadora
